HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • h1 - h6 are elements that assign headings. They start with 1 as the largest dwn to 6 the smallest.
  • ul, ol and li are for lists and list items. ul is unordered lists, ol is ordered lists and li is list items.

CSS

  • There are 3 kinds of CSS inline, internal and external.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • you connect the stylesheets in the head section using the link element.
  • CSS declarations go in a declarations block in your CSS file.
  • The CSS box model contains the margin, border, padding and content from the most outer to inner box.

Git

  • git status: checks what branch you are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScriot

  • JavaScriot allows you to add interactivity to our site and make it more dynamic.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
